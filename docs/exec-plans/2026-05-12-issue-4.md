- 시안(`../design/src/app/pages/Home.tsx`)은 "오늘의 약속" 카드를 보여주지만, 이슈는 "자주 가는 곳"(F-U02) + "최근 목적지"를 요구한다. **이슈 스펙을 우선**한다. 시안의 레이아웃 톤(플로팅 검색 바 + 풀스크린 지도 + 하단 시트형 카드)은 유지.
- 시안의 마이크(`Mic`) 버튼은 PRD에 음성 검색이 없어 제외. 검색 바 탭 시 `/search` 라우트로 이동.
